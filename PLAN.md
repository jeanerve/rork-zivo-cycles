# Zivo Cycles — Premium Fintech Savings App

## Features

- **Dashboard** showing welcome message, total savings across all cycles, discipline streak counter, and active cycle cards with animated progress bars
- **Live Growth Graph** on the dashboard that animates upward when contributions are added, with contribution markers on the timeline
- **5 Cycle Types** — Individual, Family, Community, Business, and Teen — each with unique characteristics (member lists, leaderboards, parental controls, etc.)
- **Cycle Creation Flow** — step-by-step screens to pick cycle type, set goal amount, duration, contribution frequency, and optionally invite members
- **Cycle Detail Screen** — shows progress ring/bar, contribution history, member list (for group cycles), leaderboard (community), and growth chart
- **Quick Contribution** — tap to add money with preset amounts ($25, $50, $100, $200) or custom input, with smooth graph animation and haptic feedback on success
- **Emergency Withdrawal** — request early withdrawal with discipline warning; Teen cycles show "requires parent approval" notice
- **Discipline System** — streak tracking, discipline score, missed contribution indicators, and milestone badges
- **Profile Screen** — avatar, total saved, active cycles count, streak, and badges earned
- **Notifications list** — in-app notification feed for reminders, group contributions, and goal milestones

## Design

- **Dark theme** with a near-black background (#0A0E17) and vibrant green accents (#00E676, #00C853)
- Premium glass-card style for cycle cards with subtle borders and depth
- Smooth animated progress bars that fill in real-time with a green glow effect
- Custom SVG-style growth graph with gradient fill under the line
- Rounded avatars with green status rings for group member lists
- Large bold numbers for financial amounts, clean sans-serif typography
- Haptic feedback on contributions, button presses, and milestone achievements
- Subtle entrance animations for cards and list items
- Inspired by Robinhood's clean data viz, Cash App's bold simplicity, and Revolut's premium card UI

## Screens

- **Dashboard (Home tab)** — greeting, total saved card, animated growth graph, active cycles list, discipline streak
- **Cycles tab** — browse all cycles, filter by type, create new cycle button
- **Create Cycle** — multi-step flow: select type → set details → add members → confirm
- **Cycle Detail** — full progress view, contribution history, member list/leaderboard, add contribution button, emergency withdrawal option
- **Activity tab** — notification feed and recent contribution activity across all cycles
- **Profile tab** — user stats, badges, settings

## App Icon

- Dark navy/black background with a circular green gradient ring (representing a cycle/progress)
- A subtle upward arrow or growth line inside the ring
- Clean, minimal, premium fintech aesthetic
